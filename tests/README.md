# Tests Unitaires du Projet Deck

## Structure des Tests
- `server/`: Tests du backend
- `client/`: Tests frontend
- `shared/`: Tests des utilitaires partagés

## Outils de Test
- Framework: Jest
- Coverage: Istanbul
- Types de Tests:
  ✅ Unitaires
  ✅ Intégration
  ✅ E2E (Cypress)
